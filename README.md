# Semantic Research Paper Recommender and Search Engine

A production-oriented semantic search and recommendation system for research papers, using transformer embeddings, vector databases, retrieval-augmented generation, and an API-backed demo interface.

## Setup

```bash
python -m venv .venv
.venv\Scripts\activate
pip install -r requirements.txt
copy .env.example .env
```

Fill in any optional environment variables in `.env` before running components that require external services.
